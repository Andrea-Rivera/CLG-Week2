Subscript and Superscript

The subscript text appears in a smaller font and the half character below the normal line. It is generally used to write chemical equations or chemical formulas such as H2SO4

Superscript is generally used to write mathematical equations (like x 2 + y4 = r6), footnotes, and many more.

Strong, em

Was Bold replaced by Strong?

HTML was never meant to be about styles. is semantic. Strong describes the text it surrounds (e.g., "this text should be stronger than the rest of the text you've displayed") as opposed to describing how the text it surrounds should be displayed (e.g., "this text should be bold"). Note that BOLD and ITALIC have not been deprecated in HTML5. Their new use is to semantically represent styles (or intended presentation), whereas STRONG and EM represent structure .

position: static;

An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page:

This div element has position: STATIC;

position: relative;

An element with position: relative; is positioned relative to its normal position:

This div element has position: RELATIVE;

position: fixed;

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled:

This div element has position: FIXED; bottom:0px, right:0px

position: absolute;

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed):

This div element has position: ABSOLUTE;
